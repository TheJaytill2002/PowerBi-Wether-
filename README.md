# 🌦️ Weather Analytics Dashboard

## 📌 Project Description

This project is an interactive **Weather Analytics Dashboard** built using Power BI. It uses data collected from a **Weather API** to visualize real-time and forecast weather conditions for multiple cities in a user-friendly and modern interface.

---

## 🌐 Data Source

* Weather data is fetched using a **Weather API (e.g., OpenWeather API)**
* Includes real-time and forecast data
* Data is processed and transformed using Power Query

---

## 📊 Key Features

* Multi-city weather tracking
* Current temperature and conditions
* 7-day forecast analysis
* Humidity, wind speed, pressure, visibility
* UV Index and precipitation levels
* Air Quality Index (CO, NO2, SO2, O3)
* Sunrise and sunset timings

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **Power Query** (Data Cleaning & API integration)
* **DAX** (Calculations & Measures)

---

## 🎯 Objective

To convert raw API weather data into meaningful insights using interactive dashboards and effective data visualization techniques.

---

## ▶️ How to Use

1. Open `.pbix` file in Power BI Desktop
2. Update API key (if required)
3. Refresh data
4. Explore dashboard
